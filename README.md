# SlashID (slashid)

SlashID is a developer-first identity platform providing REST APIs for passwordless authentication, multi-factor authentication, passkeys, and comprehensive user management across web and mobile applications. The platform includes three modules — Identity Management, Identity Protection, and Gate — covering authentication flows, security event monitoring, and identity-based API traffic authorization. SlashID supports magic links, biometrics, TOTP, SSO, OAuth2/OIDC, and RBAC, with global availability and an event-driven architecture designed for high-throughput production workloads.
